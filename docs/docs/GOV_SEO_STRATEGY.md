## 3. On-Page SEO Templates

### Title Tag Templates
| Trang | Format | Ví dụ |
|---|---|---|
| **Home** | `{Keyword chính} | {Brand}` | "Thiết Kế & Thi Công Nội Thất Trọn Gói \| XANH - Design & Build" |
| **About** | `Về {Brand} - {USP}` | "Về XANH - Giải Pháp Nội Thất Minh Bạch & Bền Vững" |
| **Portfolio** | `Dự Án {Type} | {Brand}` | "Dự Án Nội Thất Thực Tế \| XANH - Design & Build" |
| **Portfolio Detail** | `{Tên dự án} - {Type} {Location} | {Brand}` | "Biệt Thự Anh Hoàng - Nội Thất Biệt Thự Nha Trang \| XANH" |
| **Blog** | `Tin Tức & Cẩm Nang Xây Dựng | {Brand}` | — |
| **Blog Post** | `{Tiêu đề bài viết} | {Brand}` | "Chi Phí Xây Nhà Phố 2026: Hướng Dẫn Chi Tiết \| XANH" |
| **Green Solution** | `{USP} | {Brand}` | "Giải Pháp Nội Thất Xanh - Bền Vững & Minh Bạch \| XANH" |
| **Contact** | `Liên Hệ Tư Vấn Miễn Phí | {Brand}` | — |

> **Quy tắc:** Title < 60 ký tự. Keyword chính ở đầu. Brand ở cuối.

### Meta Description Templates
| Trang | Format |
|---|---|
| **Home** | "XANH - Design & Build: Thiết kế & thi công nội thất trọn gói tại Nha Trang. Cam kết 100% không phát sinh chi phí. Nhận dự toán miễn phí →" |
| **Portfolio Detail** | "Khám phá {Tên dự án}: {Diện tích}m², {Type} tại {Location}. Concept 3D sát thực tế {X}%. Xem Before/After →" |
| **Blog Post** | "{2 câu đầu bài viết}... Đọc thêm tại XANH →" |

> **Quy tắc:** Meta < 160 ký tự. Có CTA. Unique cho mỗi trang.

## 6. Internal Linking Strategy

### Link Map
```
Home ────────────► Portfolio (CTA: "Xem dự án")
  │                    │
  ├──► Estimator ◄─────┤ (CTA: "Dự toán")
  │                    │
  ├──► Green Solution  │
  │                    │
  └──► Blog ───────────┘ (Related: dự án tương tự)
        │
        └──► Contact (CTA mọi bài: "Tư vấn miễn phí")
```

### Quy tắc internal linking cho Blog
- Mỗi bài blog: **ít nhất 2 internal links**
  - 1 link đến Portfolio (dự án liên quan)
  - 1 link đến Estimator hoặc Contact (CTA)
- Anchor text: Descriptive, chứa keyword (KHÔNG dùng "click here")
- Related posts: Auto-suggest 3 bài cùng danh mục

## 7. Content Calendar (Blog)

### Tần suất
| Giai đoạn | Tần suất | Mục tiêu |
|---|---|---|
| Pre-launch | 5-10 bài có sẵn | SEO foundation |
| Tháng 1-3 | 2 bài/tuần | Build authority |
| Sau 3 tháng | 1 bài/tuần | Maintain |

### Topic Rotation (4 tuần)
| Tuần | Danh mục | Ví dụ topic |
|---|---|---|
| Tuần 1 | 📘 Kinh Nghiệm Xây Nhà | "5 Sai Lầm Khi Xây Nhà Lần Đầu" |
| Tuần 2 | 🧱 Vật Liệu Xanh | "Gạch AAC vs Gạch Đỏ: So Sánh Chi Tiết" |
| Tuần 3 | 🌿 Xu Hướng | "Xu Hướng Nội Thất [Năm]: Minimalism & Bền Vững" |
| Tuần 4 | 👷 Nhật Ký Xanh | "Nhật Ký Thi Công: Biệt Thự Anh Hoàng — Tuần 8" |

### Blog Post Checklist (Mỗi bài)
- [ ] Title tag < 60 ký tự, chứa keyword chính
- [ ] Meta description < 160 ký tự, có CTA
- [ ] H1 = tiêu đề, H2/H3 hierarchy đúng
- [ ] Featured image (WebP, alt text có keyword)
- [ ] Min 800 từ
- [ ] 2+ internal links (Portfolio + CTA)
- [ ] 1+ external link (nguồn uy tín)
- [ ] Table of Contents (tự động)
- [ ] Schema `Article`
- [ ] Danh mục đúng (1 trong 4)
- [ ] Share social sau publish

## 8. Image SEO

| Hạng mục | Quy tắc |
|---|---|
| File name | `thiet-ke-noi-that-biet-thu-nha-trang.webp` (keyword, lowercase, dấu gạch) |
| Alt text | Mô tả chính xác + keyword tự nhiên. VD: "Phòng khách biệt thự hiện đại tại Nha Trang — XANH Design & Build" |
| Title attr | Tùy chọn, ngắn gọn |
| Dimensions | Luôn set `width` + `height` (tránh CLS) |
| Format | WebP ưu tiên, JPEG fallback |
| Compression | Smush auto-compress |

## 9. Technical SEO Checklist

- [ ] Sitemap.xml tự động (LiteSpeed / WP native)
- [ ] robots.txt cho phép crawl (trừ admin, wp-includes)
- [ ] Canonical URLs trên mọi trang
- [ ] Hreflang (nếu có đa ngôn ngữ — Phase 2)
- [ ] Open Graph tags cho social sharing
- [ ] Twitter Cards meta tags
- [ ] Mobile-friendly test pass
- [ ] Core Web Vitals pass (LCP < 2.5s, CLS < 0.1)
- [ ] HTTPS toàn site + redirect HTTP → HTTPS
- [ ] No duplicate content (canonical)
- [ ] 404 monitoring (Search Console)
- [ ] Redirect 301 cho URL thay đổi
