## Quick Start cho AI Agent

1. **Đọc trước:**
   - `CORE_AI_CONTEXT.md` — Brand rules, file map, do's & don'ts
   - `CORE_PROJECT.md` — Project overview, sitemap, tech stack
   - `CORE_ARCHITECTURE.md` — Theme structure

2. **Khi code theme:**
   - `ARCH_DESIGN_TOKENS.md` → CSS variables
   - `PAGE_*.md` → Section specs cho trang cụ thể

3. **Khi viết content:**
   - `GOV_BRAND_VOICE.md` → Tone, CTA rules, từ cấm
   - `page_dev_guide/*.md` → Copywriting gốc
